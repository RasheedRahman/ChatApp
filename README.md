💬 Chat Application
A real-time one-to-one chat application built with Flutter & Firebase, supporting text and image messaging, online/offline status, light/dark mode, and user profile management.

✨ Features
🔐 Authentication — User Sign Up & Login with Firebase Authentication
👤 Profile Image — Set profile picture during Sign Up
👥 User List — View all connected users in one place
➕ Add Users — Connect with registered users by entering their email address
💬 One-to-One Chat — Real-time private messaging between users
🖼️ Image Sharing — Send and receive images in chat
🟢 Online / Offline Status — See when users are active in real time
🌙 Light & Dark Mode — Full theme support for better user experience
⚡ Real-time Sync — Instant message delivery powered by Firebase Firestore

📬 How It Works
User registers with email, password and a profile photo
After login, the home screen shows a list of connected users
Tap the + button to add a new user by their registered email
Tap any user to open a private chat
Send text messages or images in real time
Online/Offline status updates automatically
